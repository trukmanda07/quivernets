Expanding the Matrix Expression

Step 1: Use Matrix Properties

Start with:
$f(W) = \|y - XW\|_2^2$

Rewrite as dot product:
$f(W) = (y - XW)^T(y - XW)$

Distribute transpose:
$= (y^T - W^TX^T)(y - XW)$

FOIL (multiply out):
$= y^Ty - W^TX^Ty - y^TXW + W^TX^TXW$

Step 2: Simplify

Since $W^TX^Ty$ and $y^TXW$ are scalars and equal:

$f(W) = y^Ty - 2y^TXW + W^TX^TXW$
