Aturan Update

Bentuk umum:

$\mathbf{x}_{new} = \mathbf{x}_{old} - \varepsilon \nabla_\mathbf{x}f(\mathbf{x}_{old})$

Untuk weight linear regression:

$W_{new} = W_{old} - \varepsilon \nabla_W f(W_{old})$

dimana $\varepsilon$ adalah learning rate
