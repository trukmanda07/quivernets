Penjelasan Notasi Matrix

Pengaturannya

$\text{SSE} = \|y - XW\|_2^2$

$X \in \mathbb{R}^{m \times n}$

Matrix data input
$m$ = observasi
$n$ = fitur

$y \in \mathbb{R}^m$

Vektor output
$m$ = observasi
Nilai aktual

$W \in \mathbb{R}^n$

Vektor bobot
$n$ = fitur
Parameter yang dicari

$\|\cdot\|_2^2$ = Squared L2 Norm

Nama mewah untuk: "kuadratkan setiap elemen, lalu jumlahkan"
$\|v\|_2^2 = v_1^2 + v_2^2 + \cdots + v_n^2$
